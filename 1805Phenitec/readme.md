# 2018/5 フェニテック相乗り試作データ

2018/5のフェニテック相乗り試作で、各設計者から提出された元データ。
GDS形式のデータは、Gladeでimportするか、[GDS Viewer](https://www.vector.co.jp/soft/winnt/business/se243021.html
)や[Klayout](https://www.klayout.de/)等のGDS表示アプリで確認できます。

- OSU05 : QflowのOSU05ライブラリを使って、fa.vから生成されたGDS
- OpenRule1um : [OpenRule1um](https://github.com/MakeLSI/OpenRule1um)のスタセルを使って設計した回路
- Hibikino : [北九州ひびきの2umルール](https://github.com/MakeLSI/IP)での設計データ
- fig : イラスト
